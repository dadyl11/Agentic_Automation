Moduł 1 — Podstawy automatyzacji
Zadanie 1 — Imię jednorożca
Temat: Automatyzacja przeglądarki + pobieranie danych od użytkownika

Opis zadania

  1. Pobierz od użytkownika imię oraz miesiąc urodzenia (lista: January; February; March; April; May; June; July; August; September; October; November; December)
  2. Wejdź na stronę: https://www.rpasamples.com/findunicornname
  3. Wpisz pobrane imię i wybierz pobrany miesiąc
  4. Wyświetl wygenerowane imię jednorożca w Message Box

Wskazówki

  • Do pobierania danych od użytkownika z listą wyboru użyj aktywności Input Dialog z opcją OptionsString
  • Do otwierania przeglądarki użyj Open Browser lub Use Application/Browser
  • Do wyboru miesiąca z listy rozwijanej użyj Select Item
  • Do pobrania wygenerowanego wyniku ze strony użyj Get Text
Zadanie 2 — Kalkulator (dynamiczne selektory)
Temat: Automatyzacja aplikacji desktopowej + dynamiczne selektory

Opis zadania

  1. Pobierz od użytkownika dwie liczby z przedziału 1–9
  2. Wykonaj dodawanie tych liczb w aplikacji Kalkulator (Windows)
  3. Pobierz wynik wyświetlony w kalkulatorze
  4. Wyświetl wynik w Message Box

Wskazówki

  • Użyj Input Dialog dwukrotnie (dla każdej liczby)
  • Otwórz Kalkulator przez Open Application lub Use Application/Browser
  • Klikanie cyfr wykonaj przez aktywność Click — spróbuj użyć dynamicznego selektora (zamiast tworzyć osobny Click dla każdej cyfry, sparametryzuj selektor zmienną)
  • Do pobrania wyniku użyj Get Text
Klucz do zadania: Dynamiczny selektor to selektor zawierający zmienną, np.:
[string.Format("<uia automationid='num{0}Button'/>", str_liczba1)]
Zadanie 3 — Dni tygodnia
Temat: Praca z plikiem tekstowym + tablica + pętla + warunek

Opis zadania

  1. Przeczytaj tekst z pliku Tydzien.txt
  2. Podziel pobrany tekst na poszczególne dni tygodnia
  3. Wypisz każdy dzień (używając Log Message) w pętli
  4. Jeżeli dniem tygodnia będzie "Piątek" — wyświetl dodatkowo Message Box: "Już jutro będzie weekend!"
  5. Bonus Sprawdź w Google, jaka będzie pogoda w poszczególne dni

Wskazówki

  • Użyj Read Text File do odczytu pliku
  • Do podziału tekstu użyj metody .Split(";") na zmiennej typu String — wynik to tablica String[]
  • Do iteracji po tablicy użyj For Each (typ elementu: String)
  • Warunek sprawdzaj przez aktywność If

Zawartość pliku Tydzien.txt

Poniedzialek;Wtorek;Sroda;Czwartek;Piatek;Sobota;Niedziela
Zadanie 4 — Kalkulator w Flowchart
Temat: Flowchart + FlowSwitch + FlowDecision

Opis zadania

  1. Korzystając z Flowcharta, stwórz prosty kalkulator
  2. Pobierz od użytkownika 2 liczby całkowite i wybrane działanie matematyczne (dodawanie; odejmowanie; mnozenie; dzielenie)
  3. Wykonaj obliczenia i wyświetl wynik w Message Box
  4. Jeśli wynik jest większy od 10 — wyświetl dodatkowy Message Box z taką informacją

Wskazówki

  • Utwórz nowy projekt jako Flowchart (nie Sequence)
  • Użyj FlowSwitch do rozgałęzienia na 4 działania matematyczne
  • Użyj FlowDecision do sprawdzenia warunku wynik > 10
  • Przy dzieleniu użyj zmiennej typu Double i konwersji cdbl(), aby wynik mógł być niecałkowity
Moduł 2 — Excel i Data Table
Ćwiczenie wprowadzające — Data Table
Temat: Podstawowe operacje na DataTable (wspólnie z prowadzącym)

Opis ćwiczenia

  1. Utwórz DataTable z kolumnami: Imie, Nazwisko, Wiek i dodaj kilka wierszy
  2. Dodaj nową kolumnę do istniejącej tabeli
  3. Iteruj po wierszach pętlą For Each Row i loguj dane
  4. Usuń dodaną kolumnę
  5. Posortuj tabelę według kolumny Nazwisko
  6. Przefiltruj — zostaw tylko wiersze gdzie Wiek > 20
  7. Zapisz wynikową tabelę do pliku Excel

Kluczowe aktywności

Build Data Table  ·  Add Data Column  ·  Add Data Row  ·  For Each Row  ·  Remove Data Column  ·  Sort Data Table  ·  Filter Data Table  ·  Write Range

Ćwiczenie wprowadzające — Excel
Temat: Read Range + dynamiczna nazwa pliku (wspólnie z prowadzącym)

Opis ćwiczenia

  1. Wczytaj plik Excel (Read Range Workbook) do zmiennej DataTable
  2. Pobierz aktualną datę i sformatuj ją do postaci dd-MM-yyyy
  3. Zapisz wczytaną tabelę do nowego pliku o nazwie "Nowy plik-[data].xlsx"
Wskazówka: Do formatowania daty użyj wyrażenia Now.ToString("dd-MM-yyyy")

Temat: Filtrowanie + obliczenia + zapis raportu

Opis zadania

  1. Przefiltruj plik Produkty.xlsx — zostaw tylko rekordy, gdzie ilość jest większa niż 5

Zadanie 2 Excel — Kursy walut NBP
Temat: Automatyzacja przeglądarki + data scraping + dynamiczny zapis

Opis zadania

  1. Wejdź na stronę https://nbp.pl/statystyka-i-sprawozdawczosc/kursy/tabela-a/ i pobierz datę odświeżenia kursów walut
  2. Pobierz całą tabelę A kursów walut ze strony
  3. Wyświetl (w logu) kurs Euro
  4. Zapisz pobraną tabelę A do pliku "Kursy-tabela-A-[data odświeżenia].xlsx"

Wskazówki

  • Do pobrania daty z tekstu strony użyj Get Text, a następnie .Split(" ").Last — wyciągnie ostatni wyraz (datę)
  • Do pobrania tabeli ze strony użyj Extract Table Data
  • Kurs Euro wyciągnij przez Filter Data Table (filtruj po nazwie waluty = "euro"), a następnie odczytaj wartość przez dt_euro.Rows(0).Item(2)

Zadanie 2 — Masowa wysyłka spersonalizowanych emaili
Temat: SMTP + reużywalny workflow + argumenty

Opis zadania

  1. Wczytaj dane z pliku "Koszulki email.xlsx" (dane klientów i informacje o zamówieniach)

  • Sprawdź jakie argumenty przyjmuje Send_SMTP_Notification.xaml (zakładka Arguments)

Zadanie 4 — Analiza zamówień z maili
Temat: IMAP + filtrowanie + raport w DataTable

Opis zadania

  1. Pobierz e-maile z zadaniemail123@gmail.com — folder Robotivity
  2. Wyfiltruj tylko e-maile z tytułem zawierającym słowo "Zamówienie"
  3. Wyświetl (w logu) ilość znalezionych zamówień
  4. Stwórz tabelę z detalami zamówień i zapisz do pliku Zamowienie email.xlsx:
    Numer zamówieniaData zamówieniaEmail zamawiającego
    …dane pobrane z maili…

Wskazówki

  • Pobierz maile przez Get IMAP Mail Messages (folder: Robotivity, top: 30)
  • Wyfiltruj przez If z warunkiem currentMailMessage.Subject.Contains("Zamówienie")
  • Numer zamówienia: currentMailMessage.Subject.Split(" ").Last()
  • Data zamówienia: currentMailMessage.DateAsDateTime.ToString("dd.MM.yyyy")
  • Adres nadawcy: currentMailMessage.From.Address
Moduł 4 — Data Scraping
Zadanie — Baza firm z CEIDG
Temat: Automatyczne pobieranie danych ze strony + filtrowanie + sortowanie

Opis zadania

  1. Otwórz stronę: https://prod.ceidg.gov.pl/CEIDG/CEIDG.Public.UI/Search.aspx
  2. Wyszukaj firmy z Wrocławia z ulicy Hubskiej
  3. Pobierz ze strony maksymalnie 50 wpisów (Nazwa firmy, URL, Przedsiębiorca, NIP, REGON)
  4. Z pobranej bazy usuń wpisy, które nie mają numeru NIP lub REGON
  5. Posortuj nazwy firm od A do Z
  6. Finalną bazę danych zapisz do pliku CEIDG wyniki.xlsx

Wskazówki

  • Użyj Type Into do wpisania miasta i ulicy w pola wyszukiwarki
  • Do pobrania tabeli ze strony użyj Extract Table Data — w opcjach ustaw MaximumResults = 50
  • Do usunięcia wpisów bez NIP/REGON użyj Filter Data Table z trybem Remove i warunkiem pustości dla obu kolumn
  • Do sortowania użyj Sort Data Table (kolumna: Firma przedsiebiorcy, kierunek: Ascending)
